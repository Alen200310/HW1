## 說明程式與內容
### git ***add*** 檔案名稱:  
添加到暫存區  
___
### git ***commit***:  
檔案提交到客戶端伺服器  
___
### git ***push***:  
將檔案推送到伺服器倉庫  
___
### git ***pull***:  
從伺服器端倉庫拉取最新資料與客戶端資料**合併**
___
### git ***branck***新增分支名稱:  
建立分支  
___
### git ***checkout*** 分支名稱:  
切換到另一條分支上
___
### git ***merge*** 分支名稱:  
目前所在分支與指定分支合併
